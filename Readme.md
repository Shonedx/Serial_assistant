# 串口助手（电机控制上位机）- 核心需求 & 打包说明
## 一、核心功能需求
### 1. 串口基础能力
- 串口配置：支持波特率、串口号、数据位、停止位、校验位设置，提供打开/关闭串口操作
- 收发功能：独立收发区域（可手动调整大小），原样打印下位机数据；支持清除收发区域信息
- 格式切换：收发均支持HEX/文本模式，文本模式可切换GBK/UTF-8编码

### 2. UI 设计规范
- 基于Python开发轻量美观的图形界面，整体窗口尺寸偏大
- 侧边栏切换窗口（串口助手/PID参数/视觉设置），图标简洁优雅、大小自适应窗口
- 中文字体使用楷书，文本框支持背景色/文字色自定义调整

### 3. 参数调整功能（独立窗口）
- 自定义参数组件：可添加数据名称、协议格式，支持滑杆（可自定义最大值/最小值）+ 输入框（自适应大小，超出内容滚动显示）调整参数
- 发送模式：
  - 手动模式：点击发送，修复滑杆调整时自动发送数据的bug
  - 自动模式：可自定义发送间隔（参数+单位）、发送次数（0=无限次）
- 新增特定格式文本发送组件，内置下位机反馈显示区

## 二、打包命令

pyinstaller -F -w -n "串口助手_极客翔" --clean --collect-all customtkinter --collect-all serial serial_assistant.py
